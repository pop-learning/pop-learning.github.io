HTML Email Restrictions


Email clients do not offer the same level of HTML and CSS support that a browser does. While browsers have advanced considerably since the 1990's, email clients have made little progress. Poor support for web standards results in inconsistent rendering of HTML email.

JavaScript

Javascript is commonly used in Phishing emails and is typically disabled for security purposes. Using JavaScript can result in email clients treating your email as spam, causing deliverability issues.
