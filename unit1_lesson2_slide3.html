Coding for HTML Email

Use Inline CSS


There are three ways to add CSS to a HTML document: External CSS, internal CSS, and inline CSS. Inline CSS will take priority over the email client styles. Some email clients strip out CSS from the head of your HTML document, so using inline CSS will preserve your styles.

There are numerous online tools that will automatically inline CSS from an external stylesheet. Poppulo's Notessa allows you to add your CSS to an external stylesheet and these CSS styles will then be automatically inlined to the HTML.
