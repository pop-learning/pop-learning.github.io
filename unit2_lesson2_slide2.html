Gmail rendering issues and solutions

Problem: Extra spacing under images

Larger images are often brought into an email in slices rather than one whole image. The example image on the right demonstrates how Gmail can add unwanted spacing under each image slice.

Solution: Apply display: block as a CSS style on the image.


#image_id_name img{
     display: block;
}

For images shorter than 17px in height, apply line-height: 1px; to the <td> cell as an additional CSS style.

#image_id_name td{
     line-height: 1px;
}
