Responsive Mobile Design

Do all mail apps support Responsive Design?

iPhone and Android native mail apps both support media queries. However, on newer Android operating systems Gmail was implemented as the default mail app.

As of July 2017 Gmail is in the process of updating it's clients to support media queries.

The Samsung mail app offers partial support for media queries; it will scale fonts and images, but will not alter the layout to stack content.

The Windows mail app does not support media queries at present. Instead the desktop view will be displayed. This can result in a scaled-down desktop view and the reader having to pinch the screen to zoom in.

On the right of the screen you can see a comparison of a responsive view (left) and non-responsive view (right) of the same email. Scroll over these to see the differences in each layout.
