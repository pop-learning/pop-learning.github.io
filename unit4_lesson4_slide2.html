Media Queries and Breakpoints

What are Media Queries?

Media queries are used to apply CSS styles when an email is opened on platforms such as tablets or mobile devices. A media query consists of three parts: the media type, an expression, and the style rules contained within the media query itself.

There are four different media types: all, print, screen, and speech. The media type dictates what media the rules will be applied to.

Media queries for email use the @screen media type. The expression sets certain parameters, e.g. to target screens of certain widths. Common expressions are:
  • max- and min-width
  • max- and min-device-width
  • device-pixel-ratio
