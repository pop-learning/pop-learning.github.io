Media Queries and Breakpoints

The use of !important

Media queries are used to override inline or default CSS styles. CSS styles placed within the Media Query's curly brackets must be followed by !important.

@media screen and (max-width: 550px;) {
      td {
            padding: 20px !important;
            font-size: 15px !important;
      }
}

Applying !important to a style will force it to override any other styles that would normally take precedence.

In the example on the right of the screen the CSS is more specifically targeted for the colour green. However, by applying !important to the colour red CSS this means that red will be the dominant style.
