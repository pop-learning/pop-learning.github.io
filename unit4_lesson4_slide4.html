Media Queries and Breakpoints

What are breakpoints?

Media queries target certain viewport widths known as breakpoints. Early
thinking was to specify breakpoints for multiple mobiles devices of slightly
varying widths.

/* iPhone 6+ ----------- */
@media only screen and (min-device-width: 414px) and (max-device-height: 736px)
and (orientation : landscape) and (-webkit-device-pixel-ratio: 2) {
       /* Styles */
}

/* Samsung Galaxy S5 ----------- */
@media only screen and (min-device-width: 360px) and (max-device-height: 640px)
and (orientation : landscape) and (-webkit-device-pixel-ratio: 3) {
       /* Styles */
}

However, best-practice now recommends targeting only a limited selection of
breakpoints. Many mobile devices only have minor variations in their viewport
widths and targeting each of them is excessive.
