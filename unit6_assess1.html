Unit 1: Assessment

Test a HTML Email for Rendering Issues


Log in to the Poppulo platform on the EU server. Access the Design User Tests account and find an email called Email Test Assessment.

This email currently displays five specific rendering issues.

Complete the following steps:
  1. Send the email to the Email On Acid testing platform.
  2. Identify and list these rendering issues.
  3. Apply the appropriate CSS and amend the HTML to resolve them. Ensure that your new CSS styles are added to the end of the style sheet.
  4. Retest the email to ensure it free of rendering issues.
Once you have completed the above tasks notify your team lead. They will review the email template and the CSS that you have added. You will be asked to list the rendering issues you discovered and explain the CSS and amended HTML that you added to the email template.
